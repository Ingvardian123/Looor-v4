UraniumCraft - это скрытый плагин для Minecraft, добавляющий:
- Урановые технологии и радиацию
- Научные исследования в лабораториях

│ • Shift+ПКМ - переключение режимов                                          │
│ • Режимы: стандарт, защита, скорость, прыжки                               │
│ • Автоматическая зарядка от энергосети                                      │
│ • 95% защита от радиации                                                    │
│                                                                             │
│ ЭЛЕКТРОТРАНСПОРТ:                                                           │

│ • Режимы: стандарт, турбо, эко                                              │
│ • Дает эффекты скорости                                                     │
│                                                                             │

│ УРАНОВАЯ КАПСУЛА:                                                           │

│ • Shift+ПКМ - извлечь уран                                                  │
│ • Вместимость: 64 единицы                                                   │
│ • Безопасное хранение                                                       │
