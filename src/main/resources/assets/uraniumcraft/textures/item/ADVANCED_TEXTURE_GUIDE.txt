URANIUMCRAFT - ПРОДВИНУТОЕ РУКОВОДСТВО ПО ТЕКСТУРАМ
==================================================

ЦВЕТОВАЯ ПАЛИТРА:
================
Радиоактивные материалы: #00FF41 (ярко-зеленый), #32CD32 (лайм), #228B22 (темно-зеленый)
Энергетические элементы: #00BFFF (голубой), #1E90FF (синий), #4169E1 (королевский синий)
Квантовые эффекты: #8A2BE2 (фиолетовый), #9932CC (орхидея), #DA70D6 (розовый)
Металлические поверхности: #C0C0C0 (серебро), #708090 (серый), #2F4F4F (темно-серый)
Предупреждающие знаки: #FFD700 (золото), #FF4500 (красно-оранжевый), #DC143C (красный)

ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ:
======================
- Размер текстур: 16x16 пикселей (стандарт Minecraft)
- Формат: PNG с прозрачностью
- Светящиеся элементы должны быть яркими для видимости в темноте
- Металлические поверхности с градиентами для объема
- Анимированные текстуры: 16x256 пикселей (16 кадров по 16x16)

ЭФФЕКТЫ СВЕЧЕНИЯ:
================
- Используйте яркие цвета для светящихся элементов
- Добавьте размытие по краям для эффекта излучения
- Анимируйте пульсацию для энергетических элементов
- Создайте градиенты от центра к краям

ДЕТАЛИЗАЦИЯ:
===========
- Добавьте царапины и потертости для реализма
- Используйте шум для текстуры металла
- Создайте отражения на гладких поверхностях
- Добавьте тени для объема

СОВМЕСТИМОСТЬ:
=============
- Текстуры должны хорошо выглядеть в разных ресурспаках
- Сохраняйте читаемость при масштабировании
- Используйте контрастные цвета для важных элементов
- Тестируйте в разных условиях освещения
