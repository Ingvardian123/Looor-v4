RENDERING SPECIFICATIONS:
========================
- All textures: 16x16 pixels (except railgun: 32x16)
- Color depth: 32-bit RGBA
- Transparency: Alpha channel support
- Compression: PNG format
- Mipmapping: Enabled for performance
- Filtering: Nearest neighbor for pixel art
- Animation: Frame-based for glowing effects
- Emissive: Glow textures use emissive mapping
- Normal maps: Optional for enhanced detail
- Specular maps: Metallic surface reflection
